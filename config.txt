1. Configure C1/C2 as DHCP Servers for P1/P2

C1: 10.2.1.100 - 10.2.1.199
C2: 10.2.1.200 - 10.2.1.254

!@C1
conf t
 ip dhcp excluded-add 10.2.1.1 10.2.1.99
 ip dhcp excluded-add 10.2.1.200 10.2.1.254
 ip dhcp pool MGMTPOOL
  network 10.2.1.0 255.255.255
  default-router 10.2.1.1
  dns-server 10.2.1.1
  domain-name MGMT.COM

!@C2
conf t
 ip dhcp excluded-add 10.2.1.1 10.2.1.99
 ip dhcp pool MGMTPOOL
  network 10.2.1.0 255.255.255
  default-router 10.2.1.2
  dns-server 10.2.1.2
  domain-name MGMT.COM

!@A1
conf t
 int e0/0
 switchport mode access
 switchport access vlan 10
 end

!@A2
conf t
 int e1/0
 switchport mode access
 switchport access vlan 10
 end

!@P1
conf t
 int e0/0
 no shut
 ip add dhcp
 end

!@P2
conf t
 int e1/0
 no shut
 ip add dhcp
 end
